fun <T : Fragment> AsyncContext<T>.runOnFragmentThread(function: (T) -> Unit): Boolean
Content copied to clipboard

Execute function on the application UI thread if the underlying Fragment still exists and is not detached.

If the underlying fragment does not exist anymore or if it is detached, function will not be called.

Parameters

function

code to be executed on the main/UI thread
